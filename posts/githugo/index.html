*Truncate git history

#!/bin/bash
git checkout --orphan temp $1
git commit -m "Truncated history"
git rebase --onto temp $1 main
git branch -D temp

git prune --progress
git gc --aggressive --prune=all
git repack -a -f -F -d

Here $1 is sha-1 of the commit you want to keep and the script will create new branch that contains all commits between $1 and main and all the older history is dropped.Save this to 1.sh, use it like ./1.sh + commit sha-1.

SSH Key Authentication

In Git Bash’s command window, enter this: :

$ ssh-keygen -t rsa

Copy all of the content in the file C:\Users\{username}\.ssh\id_rsa.pub to Server.

Then type this into Git Bash: :

$ eval `ssh-agent -s`
$ ssh-add ~/.ssh/id_rsa

To test that everything works, enter this into Git Bash: :

$ ssh -T git@github.com
