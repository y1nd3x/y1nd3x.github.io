How to delete and recreate Win10/11 Recovery Partition

2026-01-17

Delete Recovery Partition

disabled the recovery function via these command

reagentc /info
reagentc /disable

then go to diskpart to delete recovery partition

sel part 4
del part override
exit

Recreate Windows Recovery Partition

go to diskpart to create recovery partition

sel part 3
extend
shrink minimum=900
create part primary
format quick fs=ntfs label="Recovery"
set id=de94bba4-06d1-4d40-a16a-bfd50179d6ac           ## set id=27  if using MBR on disk
gpt attributes=0x8000000000000001                     ## no need this line if using MBR
exit

copy winre.wim image file from your win iso to windows\system32\recovery directory

mkdir c:\tmpwim
dism /mount-wim /wimfile:"d:\sources\install.wim" /index:1 /mountdir:"c:\tmpwim" /readonly
xcopy /s /e /h /y c:\tmpwim\Windows\System32\Recovery\winre.wim C:\windows\system32\Recovery
dism /unmount-wim /mountdir:"c:\tmpwim" /discard
rd c:\tmpwim

then enable the recovery function

reagentc /enable

## using F12 hotkey to boot Recovery env directly 
Reagentc /Setreimage /Path C:\Recovery\WindowsRE /bootkey 0x8600
